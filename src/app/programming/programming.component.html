PHP University Coursework

Our course member's had to come up with 3 modules with basic information, images, categories and keywords.

All of this data was placed into a massive database and we were given a basic template.

The task:

  • Randomise the modules on the homepage
  • Implement a search function
  • Have pagination working (1,2,3,4 NEXT PAGE)
  • List view / Grid view options
  • Link categories and keywords to show the modules
  • Implement admin page to add remove and edit modules
  • Only display remove and edit options if logged in
